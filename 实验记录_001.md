# 实验 001：卫星红外超分 — DiffBIR Stage 2 免 SwinIR 微调

**日期**: 2026-05-31

---

## 实验目标

在卫星红外视频帧数据集上微调 DiffBIR Stage 2，跳过 Stage 1 SwinIR，直接用 bicubic upsampling 作为 ControlNet 的条件输入，验证这种轻量方案在红外超分上的可行性。

## 数据集

| 项目 | 值 |
|------|-----|
| 来源 | SatVideoIRSDT-extracted_frames |
| GT | 256x256, PNG, RGB, 共 1433 张 |
| LQ | 64x64, PNG, RGB, 共 1433 张 |
| 超分倍数 | 4x（64 → 256） |
| LQ 来源 | 真实红外传感器低质图（非合成 degradation） |

## 模型方案

**基线**: DiffBIR Stage 2（ControlLDM + Stable Diffusion v2.1 初始化）

**与原始 DiffBIR 的关键差异**:

1. **跳过 Stage 1 SwinIR** — 训练时直接 `F.interpolate(lq, size=gt_size, mode="bicubic")` 作为 ControlNet 的 condition 图像
2. **自定义 Dataset** — 不使用原版 `CodeformerDataset`（它会动态合成 degradation），改为直接加载成对的 GT/LQ 图像
3. **只训练 ControlNet** — UNet、VAE、CLIP 全部冻结，与原始做法一致

**修改过的文件**:

| 文件 | 修改内容 |
|------|----------|
| `train_stage1.py` | 整个文件删除，不再需要 |
| `train_stage2.py` | 移除了 SwinIR 加载，bicubic upsample 替代 `swinir(lq)` |
| `configs/train/train_stage2.yaml` | 移除了 swinir 配置块，数据集从 CodeformerDataset 切换为 SatVideoIRSDTDataset |
| `configs/train/train_stage2_v2.1.yaml` | 同上，移除 swinir 配置块 |
| `diffbir/pipeline.py` | 删除 BSRNetPipeline / SwinIRPipeline / SCUNetPipeline 三个子类，统一用 Pipeline + `apply_upsample()` |
| `diffbir/inference/bsr_loop.py` | cleaner = None，同一 Pipeline |
| `diffbir/dataset/satvideo_irsr.py` | **新增**：红外卫星视频专用 Dataset（在线随机 crop、同步 augment） |
| `run_gradio.py` | SwinIRPipeline → Pipeline 适配 |

## 训练配置

- 学习率: 1e-4
- Batch size: 8（24 GiB 显存）
- 训练步数: 30000
- 预训练权重: Stable Diffusion v2.1（`v2-1_512-ema-pruned.ckpt`）
- Noise augmentation: 0
- 输出目录: `exps/satvideo_irsr_stage2/`
- checkpoint 保存: 每 5000 步

## 启动方法

```bash
# 单卡（默认卡 0）
./train.sh

# 指定其他卡
CUDA_VISIBLE_DEVICES=2 ./train.sh

# 断点续训（修改 yaml 中 train.resume 后运行同上命令）
```
